## Подготовка к работе
1. Shift-Shift, `Project Structure`, выберите JDK 17 или 21 в качестве SDK
2. Shift-Shift, `Execute Maven Goad`, `-DskipTests=true install -f pom.xml`

## Запуск приложения

1. Выполните шаги из раздела *Подготовка к работе*
2. Shift-Shift, `UiapiApplication`
3. Нажмите кнопку `Run` слева от класса или функции `main`
